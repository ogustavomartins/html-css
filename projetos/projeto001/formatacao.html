Formatações de texto

Semântica

Semântica é o significado dos vocábulos, por oposição à sua forma. A palavra pode ter forma e significado, dessa maneira, o HTML é responsável pela semântica, e o CSS é responsável pela forma.

Negrito e Itálico

Na semâtica as tags de negrito e itálico aprensentam força e ênfase, dessa maneira o negrito é escrito pela tag <strong>, e o itálico pela tag <em>.

Marca Texto

A marcação é usada para marcar um trecho importante, é acionada pela tag <mark>.

Pequeno

Utilizado para deixar as letras pequenas, como em contratos, a tag <small> é utilizada para essa mudança.

Texto deletado

Usado para indicar que o texto está presente, mas deve ser desconsiderado pelo leitor, (tudo mentira), a tag usada é o <del>.

Texto inserido

Nesse caso, o texto inserido indica que o trecho está presente, deve ser lido e é importante, é usado a tag <ins>.

Texto sobrescrito e subcrito

Utilizado para escrever uma potência, ou uma fórmula química como: 22 e H2O, usado respectivamente a tag <sup> e a tag <sub>.

Trecho de código

Usado para transcrever um código em alguma linguagem de programação, dessa maneira as letras ficam mono-espaçadas, apresentando a mesma largura, é utilizado a tag <code>

<pre>

Juntando com a tag <pre> ele faz a formatação ficar igual a escrita no editor.

Citações

Utilizado para uma citação pequena as famosas , é utilizado a tag <q>.

<blockquote>

Para citações longas de frases e parágrafos, é usado a tag <blockquote>.

Abreviações

Utilizado para escrever o significado de alguma sigla, utiliza-se a tag <abbr>.

Texto Invertido

O texto invertido é uma tag divertida que inverte qualquer palavras ou frase, usa-se a tag <bdo> com o parâmetro rtl ou ltr.
