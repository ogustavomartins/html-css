Imagens Dinâmicas, Áudio e Vídeo

Adaptação

Existe muitos aparelhos de tamanhos diferentes atualmente, por essa razão os sites precisam se adequar a esse diferentes formatos

Imagens Flexíveis

Primeiro é necessário gerar a mesma imagem em três tamanhos diferentes uma grande, média e pequena.
Após isso, deve ser adicionado a tag <picture>, logo abaixo a tag <img> com a imagem do tamanho grande.
Por fim, imediatamente acima a tag <source> do tipo media type, selecionamos no parâmetro media: a opção max, e colocamos o tamanho do pixel da imagem.
No srcset: a imagem média e no type o formato da imagem.
Com isso, deve-se criar novamente uma tag <source>: media type imediatamente acima dessa ultima, trocando o tamanho dos pixel para o tamanho da imagem media e acrescentando no srcset a imagem pequena.
