Listas com HTML5

Conceito

As listas são extremamentes úteis e rotineiras na nossa vida, seja uma lista de supermercad, afazeres domésticos ou até uma lista online.

Listas Ordenadas

A HTML chama ordered lists todas as listas onde a ordem dos itens é algo importante. Um passo a passo, uma receita, um processo seletivo, tier lists dentre outros.
Para criar uma lista ordenada utilizamos a tag <ol> seguido da tag <li> para delimitar a lista.

  1. Primeiro <ol>
  2. Depois <li>

Type

A tag <ol> utiliza o parâmetro type, nele é possível escolher:

  • 1 - Valor padrão, cria lista numeradas: 1,2,3,4,5,...
  • A - Cria listas de letras maiúsculas: A, B, C, D,...
  • a - Cria listas de letras minúsculas: a, b, c, d,...
  • I - Cria listas com algarismos romanos em maiúsculas: I, II, III, IV,...
  • i - Cria listas com algarismos romanos em minúsculas: i, ii, iii, iv,...

Listas não Ordenadas

As listas unordered lists, chamada listas com marcadores, são as listas que a ordem não influenciará no significado da lista.
Para criar a lista é necessário usar a tag <ul> e depois a tag <li> para delimitar a lista.

  1. Primeiro a tag <ul>
  2. Depois a tag <li>

Type

As listas com marcadores também apresentam a opção de configurar o type, as opções são:

  • disc - Padrão: uma bola preta.
  • circle - Uma bola sem preenchimento.
  • square - Um pequeno quadrado preto

Lista de Definições

É como se fosse um dicionário com suas definições, está dentro da tag <dl>, cada termo é um <dt> e sua descrição é um <dd>.
