Primeiros passos com HTML5

Editores

Os melhores editores são o Visual Studio Code, Atom, Sublime, Brackets e Notepad++, dos quais o mais indicado é o Visual Studio Code.
Nele é necessário organizar em pastas: exercícios, desafios e projetos, cada um sendo enumerado quanto mais for feito.
Dentro da pasta é necessário criar o arquivo index.html que será a base do site, a parte principal.
No arquivo, somente escrevendo um ponto de exclamação e apertado enter, será criado a base do código HTML.

Tag <p>

A tag <p> introduz um parágrafo onde pode ser escrito o contéudo no site

Tag <h1>

A tag <h1> é o título principal de maior autoridade

Tag <hr>

A tag <hr> cria uma linha horizontal

Na pasta criada surgirá o index o qual pode ser aberto no navegador, assim dando forma ao primeiro site.
