Como funciona a Internet

Sinal

O computador apresenta uma linguagem única, os números binários: 0 e 1, na qual, dentro do computador o 0 significa sem onda e o 1 com onda.
A internet é uma rede gigante, na qual precisa de meios físicos para levar os sinais de um local ao outro, por isso são trafegados em cabos de fibra ótica que atravessam os oceanos. No continente, são interligados a outros sistemas de comunicação.
Contudo, esses sistemas transmitem sinais diferentes, por tal razão, é necessário o processo de modulação. Um exemplo de aparelho que faz essa modulação, é o modem das casas, o qual modula e demodula os sinais que recebe.

Cliente e Servidor

Nós, os usuários, ademais os aparelhos que utilizamos, como celulares, somos considerados clientes, os clientes estão a todo momento pedindo informações, seja sites, aplicativos, seríes, filmes, e todos são hospedados em um servidor, o qual fornece os arquivos que compõe esses serviços.

Nós

A internet funciona a baseada em um conjuto de protocolos chamados TCP/IP. O IP tem a função de indentificar os nós, nó é cada ponto que está conectado à rede. Quando você se conecta à Internet, recebe uma identificação única, chamado ENDEREÇO IP, os IPs antigos (IPv4) usavam 4 octetos, conjuto de 8 bits, totalizando 32. Os IPs modernos (IPv6) utilizam 6 octetos, totalizando 128 bits.

Acesso ao servidor

O cliente e o servidor apresentam uma identificação, um IP, para acessar um site, basicamente é necessário acessar o IP desse site, e para isso existe o DNS, eles são como grandes listas telefônicas, criando uma ligação entre o IP do site ao nome dele, por isso é necessário apenas pesquisar o nome do site desejado para encontra-lo, ao ínves de seu endereço IP.

Domínio e Hospedagem

Domínio

Quando um site é acessado, é precio saber uma URL, que é como um CEP digital do site, dentro da URL tem o domínio, nome único que identifica o servidor ou suas páginas. Para adquirir é necessário pagar, por um período mínimo de um ano.

Exemplos:

  • cursoemvideo.com
  • faetec.rj.gov.br
  • git.hub.io

As terminações desses domínios são chamados TLD.


Exemplos:

  • .com
  • .gov.br
  • .io
  • .edu.br

São divididos em GTLD para os TLD genéricos sem indicação de país e ccTLD com designação de páis. Existe o sub-domínio, o qual vem antes do domínio como o www., images., maps., mail., entre outros. Além desses, há o protocolo, que pode ser o HTTP:// ou o HTTPS://, o qual indica que o servidor tem segurança de criptografia de dados.

Hospedagem

No momento, o site realizado está armazenado no notebook, porém em grande escala há limitações, como: não fica ligado 24h, 365 dias ao ano, se faltar luz não tem sub-estação para manter tudo funcionando, por essas e outras razões é necessário contratar um plano de hospedagem.

Resumo

  • O computador indentifica uma linguagem: os números binários 0 e 1, os quais significam com sem onda e com onda respectivamente.
  • A internet é uma rede gigante, que precisa de meios físicos para os sinais do mundo se interligarem, o responsável por essa ligação são os cabos de fibra ótica que atravessam os oceanos, os sinais no continente são interligados a outros sistemas de comunicação.
  • Os sistemas transmitem sinais diferentes e o modem é responsável por modular e demodular os sinais
  • Aparelhos que utilizamos são clientes e os serviços que são pedidos estão hospedados em um servidor
  • O IP tem a função de identificar cada ponto conectado a rede, e cada identificação é chamado de ENDEREÇO IP
  • Para ler o ENDEREÇO IP, o DNS é utilizado, o qual interliga o endereço ao seu nome
  • Cada site apresenta uma URL, um CEP digital, dentro da URL tem o domínio que é o nome único contratado anualmente, a terminação dos domínios são chamados TLD, divididos em GTLD e ccTLD, além do procologo HTTP e HTTPS
  • A hospedagem é resposável por manter o site no ar.
