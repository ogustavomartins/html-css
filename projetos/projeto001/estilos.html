Trabalhando com estilos

Introdução às CSS

As CSS são a forma do site, todo o estilo feito.

CSS inline style

A forma mais simples de aplicar estilos é indo na própria tag e escrevendo o parâmetro style, assim a tag escolhida poderá sofrer alterações, porém, é somente a tag que o style foi escrito.

CSS internal style

Adicionando o <style> no <head> é possível escrever o tipo da tag que queira modificar, assim todas as tags desse tipo serão modificadas, porém, somente as tags do arquivo selecionado.

CSS external style

Essa é a forma mais versátil do CSS, pois ela é feita em um arquivo próprio, onde todos os arquivos desejados podem sofrer as mudanças somente importando o link dessa pasta do CSS.
