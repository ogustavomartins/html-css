Conceitos

HTML é uma linguagem de marcação, baseada em tags, ela é utilizada no conteúdo dos sites. As CSS também são uma linguagem de marcação, usada para fazer o design e o estilo dos sites, por fim, Javascript é uma linguagem de programação, utilizada para a interatividade do site. São os 3 pilares do desenvolvimento web.

Tags

Uma tag é um conjuto de palavras entre sinais de colchete angular, na qual a grande maioria apresenta abertura e fechamento de tag, porém, há exceções, algumas tags não apresentam fechamento.
A tag pode apresentar parâmetro com valor e conteúdo, uma tag sem contéudo não apresenta fechamento.
Existe inumeras tags obsoletas, tags que após a atualização do HTML, não serão mais funcionais nas futuras versões dos navegadores.

Seletores

Como visto, as CSS são as Cascading Style Sheets (Folhas de Esilo em Cascata), elas servem para configuração visual dos elementos HTML.
A configuração das CSS são realizadas através dos seletores, o qual a tag é o seletor, mais a abertura e fechamento de chaves, na qual, entre as duas apresenta a propriedade e o valor da configuração. Toda propriedade apresenta um valor e eles são separados por dois pontos.

Base

O HTML apreseenta uma estrutura base, digitada automaticamente no visual studio escrevendo exclamação e apertando enter.
Na estrutura base há a configuração de idioma, título, link para colocar estilos CSS e favicon, imagens e entre outros

Resumo

  • HTML é uma linguagem de marcação, usa tags, é o conteúdo do site.
  • CSS também é ums linguagem de marcação, resposável pelo design do site.
  • Javascript é uma linguagem de programação, usado na interatividade do site.
  • As tags são formadas por um conjuto de palavras entre colchetes angulares, apresentam abertura e fechamento elas apresentam um parâmetro com valor e conteúdo, sem conteúdo não apresentam fechamento.
  • As CSS são configuradas através de seletores, os quais são a própria tag sem os colchetes, mas com chaves. Apresentam a propriedade e o valor.
