Âncora

Os hyperlinks são como transportadores que liga de um ponto a outro na Word Wibe Web.
Para criar um hyperlink, é necessário criar uma âncora através da tag <a>, no atributo href pode-se colocar a URL do site ou o nome do arquivo em que seu site está.

target

Podemos usar o atributo target para acrescentar os seguintes valores:

  • _blank vai para uma nova janela.
  • _self continua na janela atual.

rel

O rel serve para dizer qual a natureza do link, ele pode ter os seguintes valores:

  • next indica que é um link que leva para a próxima parte do documento.
  • prev indica que o link volta para parte anterior do documento.
  • external indica que é um link para outro site que não faz parte do site atual.
  • author indica que o link que leva para o site do autor do artigo atual.
